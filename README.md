## ✨ Présentation

Bienvenue sur le nouveau site officiel du **Futsal Drancy**. Ce projet est une modernisation complète de la présence numérique du club, visant à offrir une expérience utilisateur "Elite" sur tous les supports (desktop et mobile).

Le site a été conçu pour refléter l'ambition et l'identité du club : **"Un État d'Esprit, une Attitude"**.

🔗 **Voir le site en direct** : [https://tharsananarul.github.io/futsal-drancy/](https://tharsananarul.github.io/futsal-drancy/)

---

## 🚀 Fonctionnalités

- 📱 **Mobile-First & Ultra-Responsive** : Navigation fluide avec menu tiroir optimisé pour smartphones.
- 🎨 **Design Premium** : Esthétique moderne utilisant les couleurs officielles (Navy & Gold).
- 📰 **Actualités & Recrutement** : Système de news intégré avec sections dédiées aux bénévoles et futurs coachs.
- 👕 **Boutique Officielle** : Présentation des équipements de la saison 2025-2026.
- 📅 **Calendrier & Résultats** : Suivi en direct des performances de l'équipe première.
- ✍️ **Inscriptions en ligne** : Accès simplifié aux démarches administratives pour la nouvelle saison.

## 🛠️ Technologies Utilisées

- **Frontend** : [React](https://reactjs.org/) + [Vite](https://vitejs.dev/)
- **Animations** : [Motion](https://motion.dev/)
- **Styling** : CSS Moderne & Tailwind CSS
- **Icônes** : [Lucide React](https://lucide.dev/)
- **Déploiement** : [GitHub Actions](https://github.com/features/actions) & GitHub Pages

## 💻 Installation Locale

Si vous souhaitez faire tourner le projet sur votre machine :

**Prérequis :** [Node.js](https://nodejs.org/) (version 18+)

1. **Cloner le dépôt** :
   ```bash
   git clone https://github.com/tharsananarul/futsal-drancy.git
   cd futsal-drancy
   ```

2. **Installer les dépendances** :
   ```bash
   npm install
   ```

3. **Lancer le serveur de développement** :
   ```bash
   npm run dev
   ```
   *L'application sera accessible sur `http://localhost:5173`*

---

## 🛡️ Maintenance & Déploiement

Le site est configuré pour se mettre à jour automatiquement via **GitHub Actions**. Chaque modification poussée sur la branche `main` déclenche une nouvelle compilation et un déploiement sur GitHub Pages.
